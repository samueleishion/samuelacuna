Visa Design System

Lead Developer

The Visa Design System (VDS) is an internal design system focused on serving applications with Visa-ready design mocks, design patterns, styles, and components. We've divided VDS into different verticals to support multiple technologies and methods of consumption. These verticals include Design, Web (html + css), Angular, React, and React Native. The project was also set up as an internal open-sourced project, in which many designers and developers across the company could contribute to the design system.

My role on this projects was to lead the Web library, which included pixel-perfect styles implemented via SASS from the design mocks. We'd also provide documentation with accessible HTML snippets for markup reference. In a way, this worked like an internal bootstrap with responsive behavior and with full accessibility (a11y) testing across browsers, screen readers, and other assistive technologies. The styles exported by the Web library was consumed by server-side rendered applications, our Angular library, and our React library.

My involvement on the project started since before it was fully chartered, by developing a library of basic components that could be themed, modified, extended, and/or forked to produce other component libraries for multiple projects across Visa. We would provide a default Visa theme, that would then turn into the VDS design and code libraries.

The VDS Web library was consumed, extended, and forked across many teams at Visa.
